# ECRM V1.6.3b 仪表盘经营复盘口径补齐记录

## 1. 本轮目标

V1.6.3b 聚焦仪表盘经营复盘口径补齐，目标是让线下摆摊结束后的复盘更可靠：

- 默认正常销售口径不被运营赠礼、人工赠送、其他出库污染。
- 纯非销售出库订单不再拉低正常销售订单数和客单价。
- 全部活动口径可以同时查看销售、运营活动、非经营出库和全出库成本。
- 页面新增固定概览，不做 V1.7 全局 UI 壳层改版。

## 2. 已完成内容

### 2.1 正常销售口径修正

- 正常销售口径下，订单数只统计存在销售明细的已支付订单。
- 正常销售口径下，客单价分母只使用存在销售明细的订单。
- 纯运营赠礼、人工赠送、其他非销售出库订单不会计入正常销售订单数和客单价。
- 混合订单仍可计入正常销售口径，但只按销售明细贡献销售额、成本、销量和排行。

### 2.2 订单性质概览

仪表盘新增“订单性质”概览：

- 正常销售。
- 销售 + 赠送。
- 非销售出库。
- 运营赠礼订单。

该模块用于快速判断当前日期范围内是否存在大量赠礼、人工赠送或其他出库记录。

### 2.3 非销售出库拆分

仪表盘新增“非销售拆分”概览：

- 满赠：数量 / 成本。
- 运营赠礼：数量 / 成本。
- 人工赠送：数量 / 成本。
- 其他出库：数量 / 成本。

该模块使用与仪表盘成本汇总一致的明细归一化口径，避免页面不同区域出现成本口径不一致。

### 2.4 经营成本口径增强

经营成本口径新增：

- 基础毛利。
- 基础毛利率。
- 活动后毛利率。

保留并继续显示：

- 优惠让利。
- 销售成本。
- 运营活动成本。
- 活动后毛利。
- 非经营出库。
- 全出库成本。

## 3. 口径说明

### 3.1 销售收入

销售收入只来自销售明细：

- 正常销售商品。
- 加购优惠销售商品。

非销售明细不计入销售收入：

- 满赠赠品。
- 运营赠礼。
- 人工赠送。
- 其他非销售出库。

### 3.2 成本归属

- 销售成本：正常销售和加购优惠销售商品成本。
- 运营活动成本：满赠成本 + 运营赠礼成本。
- 非经营出库成本：人工赠送成本 + 其他非销售出库成本。
- 全出库成本：销售成本 + 运营活动成本 + 非经营出库成本。

### 3.3 毛利口径

- 基础毛利 = 销售收入 - 销售成本。
- 基础毛利率 = 基础毛利 / 销售收入。
- 活动后毛利 = 基础毛利 - 运营活动成本。
- 活动后毛利率 = 活动后毛利 / 销售收入。
- 销售收入为 0 时，毛利率显示为 0%。

## 4. 测试情况

本轮新增和调整了仪表盘领域层与页面测试，覆盖：

- 正常销售口径订单数与客单价不被纯非销售订单污染。
- 全部活动口径下订单性质、满赠、运营赠礼、人工赠送、其他出库的数量与成本拆分。
- 页面可见订单性质、非销售拆分、基础毛利率和活动后毛利率。
- 日期范围、统计口径切换、刷新、空态和加载失败场景保持可用。

最终验证结果以本轮收尾时命令输出为准。
