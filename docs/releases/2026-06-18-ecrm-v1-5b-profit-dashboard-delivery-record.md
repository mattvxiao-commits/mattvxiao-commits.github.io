# ECRM V1.5b 毛利仪表盘交付记录

## 交付范围

- 仪表盘新增毛利概览，展示销售额（有成本快照）、成本、毛利、毛利率和赠品成本。
- 仪表盘新增 SKU 毛利排行、SPU 毛利排行和低毛利 SKU。
- 缺少成本快照的旧订单明细不会纳入准确毛利，并在页面显示提示。
- 毛利统计沿用仪表盘时间范围：今日、昨天、近 3 天、近 7 天、自定义日期。
- 页面时间范围切换后，毛利概览和毛利排行会随当前范围重新计算。

## 统计口径

- 只统计当前范围内状态为已支付的订单明细。
- 准确毛利只纳入同时具备有效 `unitCostSnapshot`、`costTotal`、`grossProfit` 的订单明细。
- `unitCostSnapshot` 和 `costTotal` 必须为有限且非负数字。
- `grossProfit` 必须为有限数字，允许为负数。
- 赠品成本计入成本，赠品明细会降低总体毛利。
- SKU/SPU 毛利排行中，赠品明细的成本和负毛利会计入对应 SKU/SPU，但赠品数量不计入销量。
- 低毛利 SKU 只展示当前范围有非赠品销量，且毛利率低于 20% 或毛利不高于 0 的 SKU。
- 人工退款仍为订单级记录，V1.5b 不拆分到商品级毛利。

## 已知边界

- 历史订单缺少成本快照时，不会用当前商品成本补算历史毛利。
- 当前人工退款不按 SKU/SPU 拆分，不影响 SKU/SPU 毛利排行。
- 商品级退款、退货入库、换货和补差价仍留到后续版本。
