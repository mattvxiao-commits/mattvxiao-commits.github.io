# ECRM V1.5.3 README 与版本信息同步记录

## 更新目标

本次更新用于同步 GitHub 仓库首页、当前线上版本和项目版本号，确保远端展示信息与 V1.5.3 当前能力一致。

文案口径采用“稳重产品说明 + 轻度场景感 + 明确边界”：

- 面向 GitHub README 的技术、产品和维护读者。
- 保留线下摆摊、网络受限、现场售卖等真实场景。
- 避免营销化表达和过强的情绪化文案。
- 明确本地数据、离线使用、备份迁移和无云同步边界。

## 已更新

- `README.md`
  - 标题调整为 `ECRM - 轻量现场经营工具`。
  - 开头定位改为离线优先、聚焦现场商品、订单、库存、促销、备份和复盘问题。
  - 增加在线地址、当前版本、适用场景。
  - 按商品与库存、售卖与收款、促销与活动、订单与售后、统计与复盘、备份与迁移重整当前能力。
  - 更新现场模式说明：商品、设置、仪表盘和订单详情受 PIN 保护，售卖页保持可用，订单记录总览可查看。
  - 强化 IndexedDB、本地备份、无云同步、跨设备迁移和离线边界说明。
  - 压缩原有 iPad 验收清单，将详细跨设备说明指向既有 guide 文档。
- `package.json`
  - 版本号从 `0.1.0` 调整为 `1.5.3`。
- `package-lock.json`
  - 同步根包版本号为 `1.5.3`。
