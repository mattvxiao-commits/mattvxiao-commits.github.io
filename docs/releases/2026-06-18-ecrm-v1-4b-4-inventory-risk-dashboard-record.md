# ECRM V1.4b-4 库存风险仪表盘交付记录

## 1. 版本定位

V1.4b-4 在 V1.4b-3 付款方式与促销统计仪表盘基础上，补充库存风险统计能力，用于线下摆摊收摊、盘点或补货前快速查看当前商品库存状态。

本版本继续保持轻量化本地离线 MVP 方向，不新增数据库 schema，不改变商品、订单或库存流水写入逻辑。

## 2. 已完成内容

- 新增售罄 SKU 统计与仪表盘展示。
- 新增高风险 SKU 统计与仪表盘展示。
- 新增滞销 SKU 统计与仪表盘展示。
- 新增补货建议统计与仪表盘展示。
- 页面展示复用现有仪表盘紧凑列表样式，没有新增全局 CSS。

## 3. 统计口径

- 库存风险使用当前商品库存，不按日期回溯历史库存。
- 售罄 SKU 为启用且当前库存为 0 的商品。
- 低库存 SKU 为启用、当前库存大于 0，且满足以下任一条件的商品：
  - 当前库存少于 3。
  - 按“当前库存 + 当前范围售出件数”估算，当前库存剩余比例不高于 20%。
- 高风险 SKU 为当前时间范围售出件数大于等于 2，且满足以下任一条件的启用可售商品：
  - 当前库存小于等于 2。
  - 按“当前库存 + 当前范围售出件数”估算，当前库存剩余比例不高于 20%。
- 滞销 SKU 为当前时间范围售出件数等于 0、当前库存大于 0、启用且可售的商品。
- 补货建议优先展示可售售罄 SKU，其次展示高风险 SKU。
- 销量判断只统计普通售卖和优惠加购明细，不统计赠品明细。
- 仅赠品不可售 SKU 不进入高风险 SKU、滞销 SKU 和补货建议。
- 售罄 SKU 可以包含可售商品和仅赠品商品，只要商品启用且当前库存为 0。
- 低库存 SKU 不再包含售罄商品，避免与售罄 SKU 重复展示。

## 6. 已知限制

- 本版本不按日期回溯历史库存，因此历史日期范围下展示的库存风险仍基于当前库存。
- 剩余比例为当前范围估算值，计算基数为“当前库存 + 当前范围售出件数”，不等同于真实历史期初库存。
- 本版本不自动生成采购单、补货任务或库存预警推送。
- 本版本不包含成本、毛利、毛利率或库存价值统计。
- 本版本不包含 CSV 导出或可配置拖拽仪表盘。
