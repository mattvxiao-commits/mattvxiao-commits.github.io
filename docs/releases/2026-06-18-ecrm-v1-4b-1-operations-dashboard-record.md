# ECRM V1.4b-1 出库与客单仪表盘交付记录

## 1. 版本定位

V1.4b-1 在 V1.4a 时间范围底座上增加出库与客单统计，用于快速查看当前范围内实际售出、赠品发放、总出库和客单价情况。

本版本只处理出库与客单，不做 SPU 表现、支付方式、活动效果、库存风险扩展，也不做 V1.5 的成本快照、毛利和库存价值。

## 2. 已完成内容

- 新增售出件数。
- 新增赠品件数。
- 新增总出库件数。
- 新增客单价。
- 仪表盘页面新增“出库与客单”指标条。
- 新增指标随当前时间范围变化。
- 页面只消费领域层 `dashboard.operationsSummary`，不在页面重复计算统计口径。
- 小屏下“出库与客单”指标条与现有指标条一致切换为单列布局。

## 3. 统计口径

- 售出件数统计当前范围内已支付订单的 `normal` 和 `discount_addon` 明细数量。
- 赠品件数统计当前范围内已支付订单的 `gift` 明细数量。
- 总出库件数为售出件数加赠品件数。
- 客单价为当前范围实收除以当前范围已支付订单数。
- 当前范围没有已支付订单时，客单价显示 `¥0.00`。
- 实收沿用 V1.4a 口径：当前范围销售额减当前范围退款金额。
- 退款金额按 `OrderRefund.createdAt` 归属当前范围。
- 范围外订单及其明细不计入出库与客单统计。
