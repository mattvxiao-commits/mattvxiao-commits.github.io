# ECRM V1.5.4a 版本号可见性补丁记录

## 背景

`v1.5.4` 增加了 PWA 新版本提示，但产品界面内没有显示当前运行版本。用户在验收线上部署和 PWA 缓存更新时，只能通过功能差异判断版本，效率低且容易误判。

## 更新内容

- 顶部品牌区新增低干扰版本标签，例如 `v1.5.4`。
- 设置页新增“系统信息”区块，显示：
  - 当前版本。
  - 部署方式：GitHub Pages / PWA。
  - 数据存储：当前设备浏览器 IndexedDB。
- Excel 导出说明使用同一应用版本常量，不再保留旧的 `0.1.0` 硬编码。
- 版本号从 `package.json` 通过 Vite 构建常量注入，减少后续版本漂移风险。

## 范围说明

- 本补丁不改变业务数据结构。
- 不改变 PWA 更新提示逻辑。
- 不改变订单、库存、备份、退款或仪表盘统计。
- 不单独提升 package 版本号，仍属于 `v1.5.4` 验收前补丁。

## 验收重点

- 任意页面顶部可看到当前运行版本。
- 设置页可查看系统信息。
- 订单 Excel 导出说明中的系统版本与当前应用版本一致。
- PWA 更新后，用户可通过界面版本号快速确认是否已切换到新版本。
