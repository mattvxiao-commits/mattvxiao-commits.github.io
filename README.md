# ECRM - 轻量现场经营工具

离线优先——解决线下摆摊网络拥挤、网络受限，聚焦现场售卖中最常见的商品、订单、库存、促销、备份和复盘问题。

适用于个人原创、同人周边、小型文创和创意团队的轻量现场经营工具。

ECRM 不是重型 ERP，也不是面向大企业流程管理的系统。它更关注真实现场：商品多、SKU 细、活动规则容易变，网络不一定稳定，但收款、库存和订单记录仍然需要尽量对得上。

## 在线使用


## 适用场景

ECRM 适合以下场景：

- 个人原创作者、同人作者、摊位经营者。
- 2-10 人的小型文创、周边设计和创意团队。
- 展会、集市、快闪、摊位等网络不稳定或现场节奏较快的售卖环境。
- 需要同时处理商品、SKU、活动促销、手动收款、库存扣减和结束后复盘的轻量经营场景。

## 当前能力

### 商品与库存

- 商品新增、编辑、复制、停用。
- 商品图、SPU/SKU 编码、售价、成本、库存。
- 可售卖、可赠品标记，支持赠品库存登记和扣减。
- 商品列表支持按商品名称、SPU、售价和添加次序排序。

### 售卖与收款

- SPU 筛选和紧凑商品列表。
- 购物车、数量调整、暂存、清空。
- 微信/支付宝收款码展示。
- 手动确认收款后保存订单并扣减库存。
- 购物车数据会本地保存，页面刷新后仍可恢复。

### 促销与活动

- 指定 SPU 加购优惠。
- 仅买优惠 SPU 时，第 1 个原价，第 2-4 个优惠，第 5 个起恢复原价。
- 满赠按最高档生效。
- 赠品库存参与库存扣减和库存流水记录。

### 订单与售后

- 订单详情、订单作废、退款记录。
- 库存回滚、库存流水、售后状态标识。
- 订单 Excel 导出，便于盘点和复盘。
- 订单明细保存下单时的售价、成本和毛利快照。

### 统计与复盘

- 今日、昨天、近 3 天、近 7 天和自定义日期范围。
- 销售额、订单数、客单价、成本、毛利、毛利率。
- SKU/SPU 排名、促销效果、收款方式、库存风险。
- 缺少成本快照的历史订单会单独提示，不会用当前商品成本回填历史毛利。

### 备份与迁移

- JSON 全量备份和恢复。
- 商品图和收款码图片随 JSON 备份导出和导入。
- 导入备份前会提示覆盖风险。
- 订单 Excel 用于统计和复盘，不能用于恢复系统数据。

## 现场模式

现场模式用于降低现场误触和敏感信息暴露风险。

开启 4 位数字 PIN 后：

- 商品、设置、仪表盘需要解锁后进入。
- 售卖页保持可用，不影响现场开单和收款。
- 订单记录总览可展开查看。
- 订单详情、库存流水、退款等敏感信息需要 PIN 解锁后才能打开。
- 正确解锁后会临时放行 5 分钟，也可在售卖页立即重新锁定。

现场模式不是多用户权限系统，也不等同于数据库加密。它的目标是解决现场设备转向顾客扫码、误触导航、临时离手等情况下的基础隐私保护问题。

## 数据与备份

ECRM 当前不使用云数据库。商品、订单、库存、图片、收款码和设置保存在当前浏览器的 IndexedDB 中。

JSON 备份包含：

- 商品资料。
- 设置和促销配置。
- 订单。
- 订单明细。
- 库存流水。
- 退款记录。
- 商品图和收款码图片。

现场模式 PIN 不会随 JSON 备份导出。导入备份到新设备后，现场模式默认关闭，需要在目标设备重新设置 PIN。

换设备使用时，请先在原设备导出 JSON 备份，再到目标设备导入。当前版本没有云同步，多设备不会自动合并订单、库存流水和退款记录。

## 离线使用说明

ECRM 是离线优先的 PWA。部署到 HTTPS 后，可在 iPad Safari 添加到主屏幕。

离线可用的前提：

- 当前设备已经在线打开过 ECRM。
- 应用静态资源已经完成缓存。
- 浏览器没有清除该站点数据。
- 没有更换浏览器、域名或设备。

摆摊当天建议只指定一台主设备开单，其他设备作为备用或导入备份后接替使用。

更完整的 GitHub Pages、跨设备和 iPad 安装说明见：

```text
docs/guides/2026-06-18-ecrm-github-cross-device-guide.md
```

## 摆摊前检查

- 商品名称、SPU/SKU、售价、成本和库存已录入。
- 商品图、微信收款码、支付宝收款码已上传并保存。
- 可售卖商品、赠品商品和赠品库存已确认。
- 加购优惠 SPU、优惠价、优惠数量和满赠档位已配置。
- 用测试订单确认优惠金额、满赠档位、库存扣减和仪表盘统计正确。
- 开启现场模式，并确认商品、设置、仪表盘和订单详情需要 PIN 解锁。
- 导出一份 JSON 备份。
- 在目标设备上离线完成一次测试开单。

## 已知边界

当前版本暂不实现：

- 复合支付二维码。
- 微信/支付宝支付 API 或自动查账。
- 云端同步、多设备自动合并。
- 飞书多维表格式自定义仪表盘。
- 小票打印、扫码枪等外设。
- 多用户账号、角色权限或云端认证。

## 本地开发

```powershell
npm install
npm run dev
```

默认开发地址：

```text
http://localhost:5173/
```

局域网设备访问时，需要使用运行电脑的局域网 IP，例如：

```text
http://192.168.x.x:5173/
```

iPad “添加到主屏幕并离线安装”需要 HTTPS。局域网 HTTP 只能用于临时预览和调试，不能作为最终 iPad PWA 安装方案。
